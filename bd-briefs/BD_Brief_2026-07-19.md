## Section 1: 신약 트렌드 (New Drug & Pipeline Trends)

### 1-1. REVTORPYK™ (gedatolisib) — FDA 승인 (July 14, 2026)

**헤드라인**: Celcuity의 PI3K/mTOR 다중 억제제 gedatolisib이 PIK3CA 돌연변이 없는 HR+ 유방암 최초 치료 옵션으로 FDA 승인 획득

**배경**: Gedatolisib은 Class I PI3K 4개 isoform(α, β, δ, γ) 전부와 mTORC1 및 mTORC2를 동시에 억제하는 pan-PAM pathway inhibitor입니다. 기존 alpelisib(Piqray)이 PIK3CA **돌연변이** 보유 환자에게만 승인된 것과 달리, gedatolisib은 PIK3CA **wild-type**(돌연변이 없는) HR+/HER2- 유방암 환자군을 타겟합니다. 이 환자군은 전체 HR+ 유방암의 약 55%를 차지하며, 기존에 효과적인 PI3K 경로 차단제가 없었습니다.

**데이터 요약 (Phase 3 VIKTORIA-1 trial)**:
- **Arm 1** (gedatolisib + palbociclib + fulvestrant): 질환 진행/사망 위험 **76% 감소** vs. 위약
- **Arm 2** (gedatolisib + fulvestrant): 질환 진행/사망 위험 **67% 감소** vs. 위약
- 대상: 내분비 치료 1회 이상 후 진행된 PIK3CA wild-type 환자
- Safety: hyperglycemia(단일 타겟 PI3K 억제제 대비 감소), 기타 grade 3+ AE는 관리 가능 수준
- PDUFA 예정일: July 17, 2026 → **July 14, 2026에 3일 조기 승인**

**BD 관점**: PIK3CA wild-type 유방암 환자 ~55%는 지금까지 PI3K 경로 차단제가 없었습니다. 이 승인은 새로운 biomarker-driven 치료 패러다임을 열며, Breakthrough Therapy Designation 및 Priority Review를 통해 빠른 심사를 받았습니다. 라이선싱 시장에서는 **PIK3CA wild-type 유방암 대상 병용 파트너** 탐색이 활발해질 전망입니다.

**경쟁 구도**:
| 약물 | 회사 | 타겟 | 환자군 |
|------|------|------|--------|
| Alpelisib (Piqray) | Novartis | PI3Kα | PIK3CA **mutant** HR+ mBC |
| Everolimus (Afinitor) | Novartis | mTORC1 | HR+ mBC (broad) |
| Capivasertib | AstraZeneca | AKT1/2/3 | PIK3CA/AKT/PTEN altered |
| **REVTORPYK (gedatolisib)** | Celcuity | pan-PI3K + mTORC1/2 | PIK3CA **wild-type** HR+ mBC ✅ |

### 1-2. Diranersen (BIIB080) — Phase 2 CELIA 결과 발표 at AAIC 2026 (July 14, 2026)

**헤드라인**: Biogen의 항tau ASO 치료제 diranersen, Phase 2에서 Primary endpoint 미달이지만 Tau 병리 감소·인지 이익 동시 달성 — Phase 3 진행 결정

**배경**: Diranersen은 Ionis Pharmaceuticals와 공동 개발 중인 antisense oligonucleotide(ASO) 치료제로, tau 단백질을 코딩하는 MAPT mRNA를 표적합니다. 기존 amyloid 표적 치료(lecanemab, donanemab)와 달리, **tau 단백질 자체의 생산을 줄이는** 상류(upstream) 전략으로, 세포 내부와 외부의 tau를 모두 감소시킬 수 있습니다. Intrathecal injection으로 투여합니다.

**데이터 요약 (Phase 2 CELIA, 18개월)**:
- **Primary endpoint** (CDR-SB dose response): **통계적 유의성 미달** (p-value 기준)
- **60mg 용량**: CDR-SB 진행 **26% 감소** (numerically significant)
- **CSF total tau**: 모든 용량에서 **50–65% 감소** (robust biomarker effect)
- **PET 기반 brain tau pathology**: 모든 용량에서 유의한 감소 확인
- Safety: 관리 가능, 중대한 신규 안전 신호 없음
- **"First tau-directed therapy to show both tau reduction AND cognitive benefit"** — Biogen

**BD 관점**: Primary endpoint 미달에도 불구하고 Biogen이 Phase 3 진행을 결정한 것은 **biomarker 결과의 일관성** 때문입니다. Tau 감소 → 인지 이익의 연결고리가 처음으로 임상적으로 시사되었고, 이는 tau 표적 AD 시장 전체에 신뢰를 높입니다. 아직 tau 표적에서 Phase 3 승인 약물이 없으므로, **first-in-class 가치**가 존재합니다. BBB 관련 플랫폼(ABL Bio Grabody-B 등)의 가치도 재조명됩니다.

**경쟁 구도**:
| 약물 | 회사 | 메커니즘 | 단계 |
|------|------|---------|------|
| Diranersen (BIIB080) | Biogen/Ionis | MAPT mRNA-targeting ASO | Phase 3 진행 예정 |
| Semorinemab | Roche/Genentech | Anti-tau IgG4 | Phase 2 (실패 이력) |
| UCB0107 | UCB | Tau aggregation inhibitor | Phase 2 |
| Lecanemab (Leqembi) | Biogen/Eisai | Anti-amyloid | 승인 (간접 경쟁) |

### 1-3. Zoldonrasib — KRAS G12D NSCLC (AACR 2026 Updated Data)

**헤드라인**: Revolution Medicines의 KRAS G12D 선택적 억제제 zoldonrasib, NSCLC에서 52% ORR·11개월 mPFS 달성 — Merck 인수 협상으로 주목

**배경**: Zoldonrasib은 KRAS G12D 돌연변이를 선택적으로 억제하는 oral tri-complex RAS(ON) inhibitor입니다. KRAS 단백질이 SOS1 같은 exchange factor에 의해 GTP-loaded(ON 상태)가 될 때, cyclophilin A(CYPA)와 함께 **삼원 복합체**를 형성해 KRAS를 활성화 불가 상태로 잡아두는 novel 메커니즘입니다. KRAS G12C 억제제(sotorasib, adagrasib)와 달리, **G12D 돌연변이 특이적**입니다. G12D는 NSCLC의 ~4%, 췌장암의 ~40%에서 발견됩니다.

**데이터 요약 (AACR 2026 Phase 1 Updated, 1200mg QD)**:
- 분석 대상: 27명 (ICI + platinum 화학요법 선행, docetaxel 미사용)
- **ORR: 52%** (confirmed objective response rate)
- **DCR: 93%** (disease control rate)
- **mPFS: 11.1개월**
- Safety: 관리 가능한 profile, 주요 grade 3+ AE 없음
- FDA Breakthrough Therapy Designation: 2026년 1월 수령

**BD 관점**: 이 데이터가 Merck의 $28–32B 인수 협상의 핵심 근거입니다. KRAS는 가장 빈번한 암 돌연변이(~25% 모든 고형암)이며, G12D가 G12C보다 더 많은 cancer type에서 발견됩니다. G12C 억제제(Lumakras, Krazati)가 보여준 $수십억 시장을 G12D가 능가할 가능성이 있습니다. NSCLC 성공 → 췌장암 확장이 핵심 BD 내러티브입니다.

**경쟁 구도**:
| 약물 | 회사 | 타겟 | 상태 |
|------|------|------|------|
| Zoldonrasib (RMC-6236) | Revolution Medicines | KRAS G12D (pan-RAS) | Phase 1/2, BTD |
| Sotorasib (Lumakras) | Amgen | KRAS G12C | 승인 (NSCLC, CRC) |
| Adagrasib (Krazati) | Mirati/BMS | KRAS G12C | 승인 (NSCLC, CRC) |
| RMC-9805 | Revolution Medicines | KRAS G12D (pancreatic) | Phase 1 |

### 1-4. Deramiocel (Capricor) — CRL 해소, PDUFA August 22, 2026

**헤드라인**: Capricor의 DMD 심근병증 대상 세포치료제 deramiocel, HOPE-3 Phase 3 데이터 제출 후 CRL 해소 — 8월 22일 PDUFA 일정

**배경**: Deramiocel은 cardiosphere-derived cell(CDC) therapy로, Duchenne Muscular Dystrophy(DMD) 환자의 심장 및 골격근 기능 보존을 목표로 합니다. DMD는 X-linked 유전 질환으로, dystrophin 단백질 결핍으로 진행성 근육 소실이 발생하며, 심근병증이 주요 사망 원인입니다.

**데이터 요약 (Phase 3 HOPE-3)**:
- **Primary endpoint**: PUL 2.0 (상지 기능 측정) — **p=0.03 달성**
- **Key secondary**: LVEF (좌심실 박출계수) — **p=0.04 달성**
- Type I error-controlled 모든 secondary endpoint 달성
- **PDUFA target action date: August 22, 2026**
- CRL 사유: 2025년 FDA가 추가 효능 데이터 요청 → HOPE-3 전체 데이터 제출 후 재심사 재개

**BD 관점**: 희귀 질환(Rare Disease) DMD에서 세포치료 승인 사례가 되면, 심근 계열 세포치료 플랫폼에 대한 BD 관심이 높아집니다. Upfront premium 비율이 희귀 질환에서 더 높은 경향이 있습니다 (Orphan Drug 지정 + Price premium).

|------|------|

| 총 고정 + 근거리 | **$950M** |

| Territory | BMS: 중국/홍콩/마카오 제외 전세계; Hengrui: 중국/홍콩/마카오 exclusive |

**프로그램 구성**:
- Hengrui 발 Oncology/Hematology: 4개
- BMS 발 Immunology: 4개
- 공동 발굴·개발: 5개
- **총 13개 프로그램** (대부분 early-stage)

**왜 이 딜인가**:
- **BMS 관점**: Revlimid LOE + Opdivo/Eliquis 특허 만료(2028) 대비 pipeline 보강이 급선무. Hengrui의 R&D 속도(중국 Phase 2 → 글로벌 Phase 3 전환)와 BMS의 글로벌 commercial 역량을 결합. 단일 에셋이 아닌 **플랫폼 협력**으로 risk diversification
- **Hengrui 관점**: $50B+ 누적 라이선싱 딜을 보유한 중국 최대 out-licensor. BMS와의 딜로 미국/유럽 시장 진출 가속화 + upfront cash $950M 확보

**밸류에이션 근거**:
- 13개 프로그램 × 개별 rNPV: 각 프로그램 peak sales 가정 $500M–$2B
- Phase 1-2 단계 success rate 적용: ~10–20% overall PoS per program
- 산술적 기댓값: 13 × avg. rNPV $250M = $3.25B (보수적 추정)
- 딜 프리미엄(플랫폼 협력, 중국 R&D access, 선제 경쟁 차단) = 추가 $11.95B
- 경쟁 벤치마크: Pfizer-Innovent $10.5B (12개 프로그램) vs. BMS-Hengrui $15.2B (13개) — BMS 딜이 약 45% 프리미엄 → immunology 4개 포함(oncology보다 낮은 PoS이지만 희귀 질환 premium)

**리스크 요인**:
- **임상 리스크**: 대부분 early-stage → 역사적으로 Phase 2→3 전환율 ~30%, 최종 승인율 ~10%
- **지정학적 리스크**: Biosecure Act 등 미-중 관계 변화가 공동 R&D에 영향 가능
- **중국 규제**: NMPA 허가 타임라인 불확실성, post-approval 가격 협상
- **IP 리스크**: 공동 발굴 프로그램의 IP 귀속 분쟁 가능성

**리스크 요인**: early-stage 포트폴리오의 임상 실패 리스크, 미-중 지정학, Pfizer 내부 우선순위 조정 리스크.

## Section 4: 오늘의 BD 역량 심화 — Deal Valuation: rNPV 계산법

### 1. 오늘의 BD 스킬: [[rNPV]] (risk-adjusted Net Present Value) 계산법

### 2. 실제 사례: BMS-Hengrui $15.2B 딜 (2026년 5월 발표)

13개 프로그램, $950M near-term payment + 최대 $15.2B. "왜 $15.2B인가?" — 이 질문에 답하려면 rNPV를 직접 계산해봐야 합니다.

### 3. 개념 설명

**rNPV(risk-adjusted NPV)**는 미래 현금흐름(DCF)에 **각 개발 단계의 성공 확률(Probability of Success, PoS)**를 곱해서 리스크를 반영한 현재 가치입니다. 제약 BD에서 표준 밸류에이션 기법으로, 단순 peak sales 추정이 아닌 "실패 확률을 감안한" 기댓값을 계산합니다.

**핵심 공식**:
```
rNPV = Σ [Annual Cash Flow(t) × PoS(t)] / (1 + r)^t  - Development Cost × PoS(투자 시점까지)
```
- `PoS(t)`: t 시점까지의 누적 성공 확률
- `r`: 할인율 (biotech에서 보통 10–12%)
- `Development Cost`: 각 Phase에 드는 비용

### 4. 실전 적용 — BMS-Hengrui 딜 가상 rNPV 계산

**가정 (단일 프로그램 예시, Phase 1 항암제)**:

| 항목 | 가정 값 |
|------|---------|
| 적응증 | 혈액암 (hematology) |
| 현재 개발 단계 | Phase 1 |
| Peak Sales 추정 | $800M (중형 혈액암 시장) |
| 상업화 연도 | 2034년 (Phase 1 → 승인 ~8년) |
| 할인율 (r) | 10% |
| Royalty rate (BMS 수취) | 20% |
| 판매 기간 | 10년 (2034–2043) |

**단계별 성공 확률 (역사적 평균)**:

| 개발 단계 | 개별 성공률 | 누적 PoS |
|----------|-----------|---------|
| Phase 1 → Phase 2 | 63% | 63% |
| Phase 2 → Phase 3 | 40% | 25% |
| Phase 3 → NDA/BLA | 58% | 15% |
| NDA → 승인 | 85% | **~13%** |

**피크 이익 현재가치 (undiscounted)**:
```
연 매출 peak = $800M
BMS royalty = $800M × 20% = $160M/year
10년 cumulative = $160M × 10 = $1,600M
```

**PoS 적용**:
```
Risk-adjusted value = $1,600M × 13% = $208M
```

**시간 가치 할인** (2026년 현재 기준, 8년 후 상업화):
```
PV = $208M ÷ (1.10)^8 = $208M ÷ 2.14 = ~$97M
```

**Development cost 차감** (BMS 부담, Phase 1–3):
```
약 $150M × PoS (Phase 1 이후 단계) ≈ $150M × 13% = $20M
Net rNPV ≈ $97M - $20M = ~$77M per program (Phase 1, hematology)
```

**13개 프로그램 포트폴리오 집계 (rough estimate)**:
```
Mix: hematology 4개 (~$77M each) + oncology solid tumor 4개 (~$60M) 
     + immunology 4개 (~$45M) + 공동 발굴 5개 (~$30M)
= ($308M + $240M + $180M + $150M) = ~$878M total portfolio rNPV
```

**딜 프리미엄 분석**:
```
Portfolio rNPV estimate: ~$878M
BMS paid near-term: $950M (upfront + 2 year payments)
Deal headline: $15.2B
```

> Emma가 봐야 할 포인트: **$950M 근거리 지급이 이미 rNPV ~$878M과 비슷하거나 약간 높습니다.** 즉 BMS는 "current expected value"만큼 upfront을 지급하고, **$14.25B의 milestones는 성공 보상**입니다. 이것이 milestone-heavy deal structure의 논리입니다.

**실제 딜이 rNPV보다 높은 이유**:
1. **전략적 프리미엄**: Hengrui R&D 스피드 내재화, 선제 경쟁 차단
2. **플랫폼 가치**: 13개 프로그램 중 하나가 "unexpected winner"가 될 수 있는 optionality
3. **지식 이전**: 중국 임상 네트워크 및 규제 know-how 접근

### 5. Emma를 위한 연습

> **과제**: Pfizer-Innovent $10.5B 딜의 ADC 단일 프로그램 rNPV를 직접 계산해보세요.
> 
> 가정 조건:
> - 적응증: HER2+ 유방암 (ADC, Phase 1)
