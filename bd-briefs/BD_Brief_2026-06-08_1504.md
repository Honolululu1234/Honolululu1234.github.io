## Section 1: 신약 트렌드 (New Drug Trends)

### 1.1 FDA 규제 동향

---

**[BTD] Plixorafenib — BRAF V600E 고등급 신경교종 (CNS Oncology)**

- **한줄 요약:** FDA가 plixorafenib에 BRAF V600E 변이 고등급 신경교종(high-grade glioma) 성인 환자 대상 획기적 치료제 지정(Breakthrough Therapy Designation, BTD) 부여. Phase 1/2a 완료 데이터 기반이며, 2026년 말 단일요법(monotherapy) 재발·진행 CNS 종양 데이터 topline 발표 예정.
- **Source:** Targeted Oncology
- **BD 시사점:** CNS 종양 분야 BRAF 표적치료의 세분화(주로 소아 LGG → 성인 HGG 확장). BTD 획득은 FDA와의 협력 개발 채널 확보를 의미하며 파트너십 딜가치(deal value) 상승 요인. BMS 포트폴리오 내 신경종양학(neuro-oncology) 공백 채울 in-licensing 후보군으로 주목 가능.

---

**[BTD] Zenocutuzumab-zbco — NRG1 유전자 융합 담도암 (Cholangiocarcinoma)**

- **한줄 요약:** FDA가 절제 불가능 또는 전이성 NRG1 유전자 융합(NRG1 gene fusion) 양성 담도암 환자에 zenocutuzumab-zbco BTD 부여.
- **Source:** OncLive
- **BD 시사점:** 희귀 바이오마커(NRG1 fusion) 기반 정밀의학 적응증으로 Orphan Drug Designation 가능성 높음. 소규모 환자군 → 높은 가격 책정(pricing) 가능 → rNPV 상승 구조. 담도암 분야 신규 기전 진입으로 기존 FGFR2(pemigatinib/futibatinib) 경쟁구도에 신규 레이어 추가.

## Section 4: 오늘의 BD 역량 인사이트 — Portfolio Deal Valuation: rNPV와 Biobuck의 진실

> **오늘의 사례: BMS ↔ Hengrui $15.2B 포트폴리오 딜을 통해 배우는 Deal Valuation**

---

### 핵심 질문: "$15.2 Billion 딜 = $15.2B의 가치인가?"

정답은 **"아니오"**입니다. 이것이 FP&A 출신 BD 전문가가 회의실에서 빛나는 순간입니다.

---

### 1. Biobuck(바이오달러)의 구조 이해

제약 BD 딜에서 발표되는 숫자는 대부분 **"biobucks"** — 즉, 모든 옵션 행사 + 모든 마일스톤 달성 + 모든 상업적 성공을 가정한 이론적 최대치입니다.

**BMS-Hengrui 딜 분해:**

| 구분 | 금액 | 성격 | 수취 확실성 |
|------|------|------|------------|
| Upfront | $600M | Signing 시 즉시 지급 | ★★★★★ 확정 |
| 1주년 지급 | $175M | 1년 후 지급 | ★★★★☆ 거의 확정 |
| 조건부 2차 지급 | $175M | 2028, 조건부 | ★★★☆☆ 조건부 |
| 개발·허가·상업 마일스톤 | ~$14.25B | 임상 성공 시 분할 지급 | ★☆☆☆☆ 확률 가중 필요 |
| **합계** | **$15.2B** | | |
| **실질 확정 수취액** | **~$775M** | | 전체의 **약 5.1%** |

---

### 2. rNPV(Risk-Adjusted NPV) 계산 사고방식

```
마일스톤 rNPV = 마일스톤 금액 × 임상 성공 확률 × 할인 계수(시간가치)
```

**업계 평균 임상 성공률 (종양학 기준):**
- Phase 1 → Phase 2 진입: ~65%
- Phase 2 → Phase 3 진입: ~40%
- Phase 3 → NDA 승인: ~60%
- **Phase 1 → NDA 최종 승인: 약 5~10%**

**적용 예시 (조기 임상 단계 마일스톤):**
```
예: 승인 마일스톤 $200M × 7% 성공확률 × 할인계수(0.7) = $9.8M (rNPV)
```

실제 협상에서 양측이 합의한 **딜의 실질 rNPV는 $1~3B 수준**으로 추정됩니다.

---

### 3. BMS 입장 — Real Options 전략

- 13개 조기 임상 프로그램 = **저렴한 가격에 13개의 "옵션" 매입**
- 성공하는 1-2개 자산의 상업 가치만으로도 $950M 확정 지급 회수 충분
- 포트폴리오 분산 → 단일 자산 임상 실패 리스크 헤징
- Hengrui의 빠른 IND 신청 속도 + 중국 내 대규모 환자 모집 능력 활용

---

### 4. Hengrui 입장 — Cross-Licensing 전략

- **$950M 확정 수취:** 자체 글로벌 임상 비용 대비 확실한 현금 유입
- **Greater China 권리 보유:** 글로벌 딜에도 내수 시장(중국) 수익화 유지
- **BMS 자산의 중국 권리 확보:** in-licensor + out-licensor 양방향 파트너로 격상

---

### 5. Emma에게 드리는 FP&A → BD 전환 핵심 인사이트

> Finance 출신이 BD에서 진짜 강점을 발휘하는 순간은 바로 여기입니다.

다음 번 내부 딜 리뷰 회의에서 활용할 수 있는 질문들:

1. **"Probability-weighted milestone value는 얼마로 계산하셨나요?"**
2. **"Upfront 대비 마일스톤 비중이 이렇게 높은 이유는 무엇인가요?"**
3. **"각 마일스톤의 예상 달성 시기 timeline이 있나요? NPV 기준 현재가치로 환산해 보셨나요?"**
4. **"13개 중 몇 개가 Phase 3 진입하면 BEP(Break-Even Point)를 넘나요?"**

**결론:** Finance 백그라운드는 BD 딜 평가에서 **약점이 아니라 최강의 무기**입니다.
