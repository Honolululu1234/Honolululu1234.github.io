## Section 1: 신약 트렌드 (New Drug & Pipeline Trends)

### 1-1. BMS Mezigdomide (SUCCESSOR-2) — Phase 3, RRMM에서 PFS 두 배 이상 달성

**헤드라인**: BMS 차세대 CELMoD Mezigdomide, Phase 3 RRMM 연구에서 PFS 18개월 vs 8.3개월로 표준치료 압도 — Revlimid 이후 BMS의 최대 파이프라인 답안

**배경**: Mezigdomide는 [[CELMoD]](Cereblon E3 Ligase Modulator) 플랫폼 기반 약물로, 기존 IMiD(Lenalidomide·Pomalidomide)와 달리 종양 억제 전사인자 Ikaros·Aiolos를 더 빠르고 완전하게 단백질 분해(protein degradation)해 lenalidomide-refractory 환자에서도 효력을 유지한다. SUCCESSOR-2는 lenalidomide 및 anti-CD38 항체(예: daratumumab) 모두 사전 치료받은 Relapsed/Refractory Multiple Myeloma(RRMM) 환자에서 MeziKd(Mezigdomide + Carfilzomib + Dexamethasone) vs Kd(Carfilzomib + Dexamethasone)를 비교한 무작위 Phase 3 연구. 결과는 ASCO 2026(5월 29일~6월 2일, 시카고)에서 Late-breaking으로 발표되고 The Lancet에 동시 게재됐다.

**데이터 요약**:

| 지표 | MeziKd (실험군) | Kd (대조군) |
|------|-----------------|-------------|
| Median PFS | **18.0개월** | 8.3개월 |
| PFS 위험 감소 | **52%** (HR ~0.48) | — |
| Grade 3-4 TEAEs | 83.7% | 56.5% |
| Neutropenia (G3-4) | 61.1% | 9.1% |
| Infection (G3-4) | 34.0% | 15.6% |

*출처: ASCO 2026 LBA7506; The Lancet interim analysis (2026.06)*

**BD 관점**: SUCCESSOR-2 성공은 BMS의 CELMoD 플랫폼 전체 가치를 확정한다. Revlimid(Lenalidomide) LOE로 연간 ~$7B 매출이 2026~2028년 사라지는 BMS에게 Mezigdomide는 가장 중요한 내부 성장 드라이버다. 후속 CELMoD(Iberdomide, Golcadomide)의 라이선싱 가치도 동반 상승한다. 외부 파트너십 관점에서는 CELMoD와 시너지를 낼 수 있는 CD38 항체, BCMA ADC, 혹은 NK세포치료 기술을 가진 회사가 BMS 파트너 탐색 대상이 될 가능성이 높다.

**경쟁 구도** (RRMM):
- **J&J Carvykti** (ciltacabtagene autoleucel, BCMA CAR-T): CARTITUDE 시리즈로 early-line 확장 중
- **J&J Teclistamab** (BCMA × CD3 bispecific): MajesTEC-1/3에서 단독 및 병용 데이터 축적
- **BMS Ide-cel** (idecabtagene vicleucel, BCMA [[CAR-T]]): KarMMa-3 RRMM 승인, Mezigdomide 병용 조합 탐색 중

### 1-2. Iza-bren (Izalontamab Brengitecan) — Phase 3 이중 생존 지표 동시 충족

**헤드라인**: BMS·SystImmune 공동 개발 EGFR×HER3 bispecific [[ADC]] Iza-bren, TNBC 및 식도암 Phase 3에서 OS·PFS 모두 통계적 유의성 달성 — 단일 에셋 3개 Phase 3 동시 양성

**배경**: Iza-bren(BL-B01D1)은 세계 최초의 EGFR×HER3 이중표적 [[ADC]]다. 하나의 항체가 EGFR과 HER3를 동시에 잡아 종양세포 표면에 결합하고, Topoisomerase I 억제제 exatecan derivative(Ed-04) 페이로드를 세포 내부로 전달해 DNA 손상을 유발한다. Bystander effect로 주변 종양세포에도 페이로드가 확산되는 특성이 있다. EGFR과 HER3가 모두 높게 발현되는 상피암(TNBC, NSCLC, ESCC, 방광암 등)에서 넓은 적용 가능성을 갖는다.

**데이터 요약** (ASCO 2026 발표 및 BMS 공시):
- **TNBC**: OS·PFS 모두 통계적으로 유의미한 개선
- **ESCC**: 재발·전이 2L+ 환자 — 이중 생존 지표 충족
- **NSCLC (EGFR 변이)**: Pooled Phase I/II ORR 42.9~50%
- **방광암**: IZABRIGHT-Bladder01 Phase 2/3 진행 중

**BD 관점**: 3개 Phase 3 동시 양성은 Iza-bren이 단일 적응증 약물이 아닌 "플랫폼형 ADC"임을 입증한다. 2개 indication BLA 제출이 2026년 하반기~2027년 초 예상.

**경쟁 구도** (TNBC):
- **AstraZeneca/Daiichi Sankyo DATROWAY** (datopotamab deruxtecan, TROP2 ADC): TROPION-Breast02 1L TNBC
- **Gilead Trodelvy** (sacituzumab govitecan, TROP2 ADC): TNBC 2L+ 기승인
- **AZ/Daiichi T-DXd** (trastuzumab deruxtecan, HER2 ADC): HER2-low TNBC 영역 확장 중

## Section 4: BD 역량 심화 — [[Deal Valuation]]: rNPV 계산법

**오늘의 BD 스킬**: rNPV (Risk-adjusted Net Present Value)

**실제 사례**: Pfizer × Innovent $10.5B 딜 (2026.05.28)

**개념 설명**:

rNPV = 신약 미래 현금흐름 × 성공 확률(PoS) × 할인

```
rNPV = Σ [현금흐름 × 누적 PoS × (1/(1+r)^t)] − 개발비용 현재가치
```

**단계별 PoS (Oncology)**:
| 단계 | PoS |
|------|-----|
| Phase 1 → Phase 2 | ~63% |
| Phase 2 → Phase 3 | ~35% |
| Phase 3 → NDA | ~58% |
| NDA → 승인 | ~85% |
| **Phase 1 → 승인 (누적)** | **~13~15%** |
| **Phase 3 → 승인 (누적)** | **~49%** |

**실전 적용 (Pfizer-Innovent)**:
```
에셋 1개: $800M × 30% × 75% × 13% × 6.1 = ~$113M
12개 합산: ~$1.36B (단순 rNPV)
```

**Gap 분석 (단순 rNPV $1.4B vs 딜 $10.5B)**:
| 요인 | 설명 |
|------|------|
| ① Milestone 조건부 | $9.85B은 최대값, 실제 EV = $2~3B |
| ② Upfront만 확정 | $650M만이 실질 지출 |
| ③ Portfolio premium | 12개 동시 확보 = 다각화 + 경쟁사 차단 |
| ④ Platform 접근권 | Innovent R&D 지속 접근 (optionality) |
| ⑤ Strategic premium | 중국 R&D 속도·비용 우위 선점 |

> **핵심 교훈**: Headline value ≠ 실질 기대 지급액. "Upfront만으로도 합리적인가?"를 먼저 확인하고, milestone별 PoS를 적용한 weighted EV를 계산할 것.

**Emma 연습 과제**: GSK-Nuvalent $10.6B을 직접 rNPV로 검증
1. ClinicalTrials.gov에서 Zidesamtinib 임상 현황 확인
2. ROS1 NSCLC 시장 규모 추정 (전체 NSCLC의 1~2%)
3. Peak sales 추정 (Xalkori, Rozlytrek 참고, 보수적 $800M~낙관적 $1.5B)
4. PoS ~85% (FDA 심사 중) 적용
5. Zidesamtinib + Neladalkib rNPV 합산 후 $10.6B와 비교
