## Section 1: 신약 트렌드 (New Drug Trends)

### 🔬 FDA 규제 액션 (FDA Regulatory Actions)

---

**[BTD] Olomorasib + Pembrolizumab → NSCLC**
- **한줄 요약:** FDA가 KRAS G12C 저해제 olomorasib와 Keytruda 병용요법에 NSCLC 적응증으로 Breakthrough Therapy Designation(BTD) 부여.
- **Source/Company:** Eli Lilly (olomorasib) + MSD (pembrolizumab)
- **BD 관점:** KRAS G12C 타깃팅은 수 년간 "undruggable" 로 불렸던 영역. BTD 획득은 규제 검토 가속화를 의미하며, KRAS 경쟁(AMG 510/sotorasib vs. MRTX849/adagrasib) 구도에서 새 병용전략이 차별화 포인트가 됨. BMS 입장에서 면역항암제(IO) 병용 경쟁 지형 모니터 필요.

---

**[BTD] Plixorafenib → BRAF V600E 고등급 신경교종**
- **한줄 요약:** FDA가 성인 BRAF V600E 돌연변이 고등급 신경교종(glioma)에 plixorafenib의 BTD 부여; Phase 1/2a 및 진행 중인 Phase 2 FORTE basket trial 데이터 기반.
- **Source/Company:** Day One Biopharmaceuticals
- **BD 관점:** 신경교종은 높은 미충족 의료수요(unmet medical need) 영역. BRAF-mutated 하위 집단을 타깃하는 바스켓 전략은 소규모 적응증에서 규제 효율성을 극대화하는 접근법 — BD 딜 구조에서 indication expansion 옵션 설계 시 중요 참조 사례.

**[Phase 3] Daraxonrasib → 전이성 췌장암 (mPDAC)**
- **한줄 요약:** RASolute 302 Phase 3에서 daraxonrasib가 투자자 선택 화학요법 대비 사망 위험 **60% 감소** — ASCO 2026 Plenary Session 주요 발표.

- **한줄 요약:** SARC041 Phase 3에서 CDK4/6 저해제 abemaciclib이 진행성 역분화 지방육종 환자에서 위약 대비 개선 효과 확인.
- **Source/Company:** Eli Lilly
- **BD 관점:** CDK4/6 저해제의 유방암 외 적응증 확장 — indication broadening 전략의 대표적 성공 사례. 기존 약물의 새 적응증 확장은 라이선스 옵션 계약(option-to-license) 또는 co-development 구조에서 자주 활용.

- **Source/Company:** AstraZeneca
- **BD 관점:** 아시아 (특히 한국·중국·일본) 최고 유병률 암종인 HCC에서 IO+TACE 병용 표준치료 확립 가능성. 한국 BMS에서 시장 영향 직접 관련.

## Section 4: 오늘의 BD 역량 인사이트 — 딜 구조 분석: BMS × Hengrui $15.2B 딜을 통해 배우는 **마일스톤 설계와 rNPV 평가**

### 📚 오늘의 BD 역량: Deal Valuation — 마일스톤 구조와 rNPV 이해

---

#### 왜 오늘의 딜인가?

BMS가 자신의 고용주인데, $15.2B짜리 딜의 숫자가 어떻게 만들어지는지 아는 것은 Finance/FP&A로서 전략 기여도를 높이는 핵심입니다.

---

#### 딜 구조 해부 (Deal Anatomy)

```
총 계약가치 $15.2B 분해:

[즉시 확정 현금]
  업프런트 (Upfront)         = $600M   ← 계약 체결 시 즉시 지급
  1주년 지급                 = $175M   ← 2027년 확정 지급
  2주년 조건부 지급           = $175M   ← 2028년 조건부
  소계 (near-certain value)  = ~$950M

[불확실 마일스톤]
  개발 마일스톤 (POC, Ph2 완료 등) = X
  규제 마일스톤 (IND, NDA 등)      = Y
  상업 마일스톤 (매출 기준점)       = Z
  옵션 행사 (공동발굴 5개 프로그램) = W
  소계 = ~$14.25B ← 13개 프로그램 전부 성공 시 이론적 최대치
```

> **핵심 인사이트:** "$15.2B 딜"이라는 헤드라인은 **이론적 최대값**이지, 현재가치(Present Value)가 아닙니다.

---

#### rNPV (risk-adjusted Net Present Value) 란?

BD에서 딜 가치를 평가할 때 단순 NPV가 아닌 **rNPV**를 씁니다:

```
rNPV = Σ [현금흐름(t) × 성공확률(t)] / (1 + r)^t
```

**BMS-Hengrui 딜에 적용하면:**

| 단계 | 산업 평균 성공 확률 | 적용 예시 |
|------|-------------------|-----------|
| Preclinical → Phase 1 | ~60% | 13개 프로그램 대부분 전임상 |
| Phase 1 → Phase 2 | ~50% | |
| Phase 2 → Phase 3 | ~30% | |
| Phase 3 → Approval | ~50% | |
| **전임상 → 승인 누적** | **~9%** | |

→ 13개 프로그램 × 평균 전임상 단계 × ~9% 누적 성공률 = **기대 승인 프로그램: 약 1.2개**

#### BD 실무에서 쓰는 빠른 Peak Sales 추정법

```
Peak Sales 추정 = Target Population × 진단율 × 치료율 × 약가 × 시장 점유율

예시 (가상의 oncology 자산):
  - 미국 DLBCL 신규 환자: ~18,000명/년
  - 3L 이후 적격 환자: ~30% = 5,400명
  - 치료율: 60% = 3,240명
  - 연간 약가: $150,000
  - 시장 점유율 (경쟁 고려): 25%
  → Peak Sales ≈ $122M/년
  → rNPV (discount rate 10%, 성공확률 30%): ~$200~300M
```

→ 이 계산이 바로 딜의 **'업프런트 얼마가 적정한가'**를 결정하는 핵심 논리.
