## Section 1: 신약 트렌드 (New Drug & Pipeline Trends)

### 1-1. ASCO 2026 ADC 빅뱅: 1차 치료 TNBC에서 ADC가 표준치료 대체 선언

**헤드라인**: ASCO 2026에서 sacituzumab govitecan과 datopotamab deruxtecan이 1차 전이성 TNBC에서 화학요법 대비 우월한 PFS/OS 입증, FDA 1차 적응증 승인 완료

**배경**:
- **Sacituzumab govitecan (SG, Trodelvy)**: Gilead/Pfizer가 보유한 TROP2 타겟 [[ADC]]. Payload는 SN-38 (이리노테칸의 활성 대사체). cleavable linker가 종양 내 산성 환경에서 절단되어 인근 암세포에도 bystander effect를 발휘. 전이성 TNBC 3차 치료로 시작해 이제 1차 치료까지 적응증 확대.
- **Datopotamab deruxtecan (Dato-DXd, Datroway)**: AstraZeneca/Daiichi Sankyo 공동개발. TROP2 타겟 ADC, Payload: DXd (exatecan 유도체, topoisomerase I 억제). T-DXd(Enhertu)와 같은 Daiichi Sankyo 플랫폼 기반 — high drug-antibody ratio(DAR 8)와 cleavable tetrapeptide linker로 bystander killing 극대화.
- 두 약물 모두 TROP2를 타겟하지만 payload·linker·DAR이 달라 efficacy·safety 프로파일에 차이.

**데이터 요약**:

| 약물 | 시험명 | 환자군 | 주요 결과 |
|------|--------|--------|----------|
| SG + pembrolizumab | ASCENT-04 | PD-L1 양성 mTNBC 1차 | mPFS **11.2 vs 7.8개월** (HR 0.65, 35% 위험 감소) |
| SG 단독 | ASCENT-03 | PD-L1 음성 mTNBC 1차 | mPFS **9.7 vs 6.9개월** (38% 위험 감소) |
| Dato-DXd | TROPION-Breast02 | 면역요법 부적합 mTNBC 1차 | mPFS **10.8 vs 5.6개월**, **mOS 23.7 vs 18.7개월** (mFU 27.5개월) |

- FDA: SG를 1차 mTNBC 두 개 적응증(PD-L1 양성: SG+pembro; PD-L1 음성: SG 단독)으로 동시 승인 완료
- Dato-DXd: OS benefit 입증 → FDA 1차 치료 적응증 확장 신청 준비 중 (2026 하반기 예상)

**BD 관점**:
- ADC가 1차 치료로 올라오면서 기존 화학요법+checkpoint inhibitor 조합의 시장 잠식 가속화
- TROP2 ADC 공간에서 다음 BD 기회: (1) 내성 극복(post-ADC 옵션), (2) payload 다양화, (3) 새로운 tumor type 확장 (NSCLC 1차, HR+/HER2- 유방암)
- LegoChem, Daewoong 등 한국 ADC 플랫폼 보유사들의 TROP2 이후 차별화된 target 자산 가치 재평가 가능성

**경쟁 구도**:
1. **Trodelvy (sacituzumab govitecan)** — Gilead/Pfizer (TROP2, SN-38) → **FDA 1차 TNBC 2개 적응증 승인**
2. **Datroway (datopotamab deruxtecan, Dato-DXd)** — AstraZeneca/Daiichi Sankyo (TROP2, DXd) → **OS benefit 입증, 1차 적응증 확장 준비**
3. **BL-B01D1** — BMS/SystImmune (EGFR×HER3 bispecific ADC) → **Phase 3 PFS+OS 동시 달성 (2026년 2월)**

### 1-2. BL-B01D1: 중국산 Bispecific ADC, Phase 3에서 TNBC Dual Endpoint 달성

**헤드라인**: BMS/SystImmune의 EGFR×HER3 bispecific ADC BL-B01D1, TNBC 1차 치료 Phase 3에서 PFS·OS 모두 통과 — BMS 고형암 포트폴리오 강화

**배경**:
- BL-B01D1은 EGFR과 HER3 두 개 항원을 동시에 타겟하는 bispecific [[ADC]]. EGFR과 HER3는 모두 ErbB 계열 수용체 티로신 키나제이며 암세포 증식·생존 신호의 핵심 드라이버.
- HER3는 EGFR 단독 억제 후 발생하는 내성의 주요 escape route — bispecific 전략이 내성 극복의 열쇠.
- **[[BMS]]**는 2024년 BL-B01D1의 글로벌(중국 제외) 개발/판매권을 $8.4B 규모로 확보 (upfront $800M). SystImmune은 중국 rights 보유.

**데이터 요약**:
- Phase 3 (TNBC 1차 치료): 화학요법 대비 **PFS 및 OS 모두 통계적 유의성 달성** (2026년 2월 발표)
- 구체적 수치: 회사 발표 기준 PFS HR < 0.70, OS HR < 0.80 (정확한 수치는 ASCO/ESMO 발표 예정)
- ASCENT·TROPION 데이터와 함께 TNBC 1차 치료 ADC 트리오 구도 형성

**BD 관점**:
- BMS 입장에서 BL-B01D1 Phase 3 성공은 $800M upfront 정당화의 핵심 데이터
- NDA 제출 시 BMS는 Breyanzi(혈액암 5개 적응증)+BL-B01D1(TNBC 포함 고형암)로 oncology 포트폴리오 본격 다각화
- bispecific ADC 콘셉트 검증 완료 → 동일 bispecific 전략으로 NSCLC, 위암 등 확장 시험 파이프라인 가치 재평가

**경쟁 구도**:
1. **BL-B01D1** — BMS/SystImmune (EGFR×HER3, auristatin계 payload) → **Phase 3 성공**
2. **JSKN016** — Alphamab (TROP2/HER3 bispecific ADC) → Phase 1 진행 중, 2026 ASCO 데이터 발표
3. **HER3-DXd (patritumab deruxtecan)** — Daiichi Sankyo/AZ (HER3 단일타겟 ADC) → NSCLC FDA 승인, TNBC 확장 시험 중

### 1-3. Aspen Neuroscience ASPIRO: 파킨슨병 최초 자가 iPSC 세포치료제 12개월 긍정 데이터

**헤드라인**: 자가 iPSC 유래 도파민 뉴런 전구세포 sasineprocel, 파킨슨 환자 8명에서 12개월 안전성·운동기능 개선 확인 — Phase 3 진입 선언

**배경**:
- **Sasineprocel (ANPD001)**: 환자 자신의 세포에서 induced pluripotent stem cells (iPSC)를 생성하고, 이를 도파민성 뉴런 전구세포(dopaminergic neuron precursor cells, DANPCs)로 분화시켜 뇌 흑질(substantia nigra)에 직접 이식하는 세포 재생 치료제.
- 파킨슨병의 핵심 병태생리: 흑질의 도파민 뉴런 점진적 소실 → 운동 조절 실패. 기존 약물(레보도파 등)은 증상 완화에 그치며 신경 소실은 계속됨. sasineprocel은 소실된 뉴런을 직접 대체(replace)하는 근본적 접근.
- 자가이식(autologous)이므로 면역억제제 불필요 — 동종(allogeneic) 세포치료 대비 핵심 안전 우위.

**데이터 요약**:
- ASPIRO Phase 1/2a, 8명 환자, 12개월 추적 (2026년 3월 AD/PD 국제학회 발표)
- **안전성**: 면역억제 없이 중대한 부작용 없음, tolerability 확인
- **유효성 (탐색적)**: UPDRS Part III(운동 척도) 수치적 개선; 환자보고결과(PRO) 및 삶의 질 개선
- 뇌 내 이식 세포 생착(engraftment) MRI/PET으로 확인
- **Phase 3 계획**: 2026 하반기 시작 발표

**BD 관점**:
- 자가 iPSC 세포치료제는 [[CAR-T]]와 유사한 'personalized medicine' 패러다임 — 제조 복잡성·비용이 높지만 면역거부 문제 원천 해결
- Aspen은 현재 private 회사; Phase 3 진입 전후 IPO 또는 빅파마 파트너십/인수 타겟 가능성 고조
- 신경과학 세포치료 공간은 경쟁자 희소 → First Mover 프리미엄 기대
- 한국 iPSC/세포치료 연구 기관(차병원, 한국세포치료학회 연계 기업 등)의 유사 기술 BD 가치 벤치마킹 가능

**경쟁 구도**:
1. **Sasineprocel (ANPD001)** — Aspen Neuroscience (autologous iPSC-DANPCs) → **Phase 1/2a 12개월 긍정, Phase 3 준비**
2. **BIIB122 (DNL151)** — Biogen/Denali Therapeutics (LRRK2 kinase 억제 소분자) → Phase 2 Luma 시험 완료 단계
3. **Prasinezumab** — Roche/Prothena (anti-α-synuclein 단클론 항체) → Phase 2b 진행 중 (조기 PD 대상)

### 1-4. BMS milvexian: ACS Phase 3 조기 종료 — Factor XIa 전략 AF·STROKE로 집중

**헤드라인**: BMS·J&J 공동개발 milvexian(Factor XIa 억제제), ACS Phase 3(Librexia ACS) 조기 종료 — 심방세동·뇌졸중 시험은 계속

**배경**:
- **Milvexian**: 경구용 Factor XIa 억제제. 기존 항응고제(와파린, DOAC)와 달리 혈전 형성의 "접촉 활성화 경로"만 차단해 출혈 위험을 줄이면서 항혈전 효과를 내겠다는 novel mechanism.
- Factor XIa는 심방세동, 뇌졸중, 심근경색 관련 혈전 형성에 관여하나, Factor Xa·IIa(트롬빈)와 달리 정상 지혈에서의 역할이 작아 출혈 부작용이 적을 것으로 기대.

**데이터 요약**:
- **Librexia ACS (급성 관상동맥증후군)**: IDMC(독립 데이터 모니터링 위원회) 중간분석에서 "주요 efficacy endpoint 달성 불가" 판정 → BMS·J&J 공동으로 시험 조기 종료
- 새로운 안전 우려는 없음 (safety profile 일관)
- **Librexia AF** (심방세동): 계속 진행, 2026 하반기 top-line 예정
- **Librexia STROKE** (뇌졸중 2차 예방): 계속 진행, 2026 하반기 top-line 예정

**BD 관점**:
- ACS 실패는 BMS CV 파이프라인에 단기 부정적 신호 — 그러나 AF·STROKE 두 개 indication이 남아 있어 완전한 프로그램 종료는 아님
- Factor XIa 공간 전체에 불확실성 증가: Abelacimab(anthos/J&J), Asundexian(Bayer) 등 경쟁 Factor XI/XIa 억제제들도 영향받을 수 있음
- BMS 입장: CV 파이프라인 약화 시 외부 BD(in-licensing/M&A)로 gap 보완 필요성 증가

**경쟁 구도**:
1. **Milvexian** — BMS/J&J (Factor XIa, 소분자) → ACS 실패, AF·STROKE 진행 중
2. **Asundexian (BAY 2433334)** — Bayer (Factor XIa, 소분자) → AF 대상 OCEANIC 프로그램 진행 중
3. **Abelacimab** — anthos Therapeutics (Factor XI 단클론 항체) → AF Phase 3 진행 중

## Section 2: 글로벌 BD 딜 분석 (Global Pharma BD Deal Analysis)

### 2-1. Pfizer × Innovent Biologics: $10.5B 메가딜 — 12개 종양학 프로그램 글로벌 협력

**딜 개요**:
Pfizer(미국)와 중국 Innovent Biologics가 ADC 및 multi-specific antibody 중심 12개 oncology 프로그램에 대한 전략적 글로벌 공동개발 파트너십 체결. 2026년 5월 29일 발표, Q3 2026 클로징 예정.

**딜 구조**:
- **Upfront**: **$650M** (Innovent 수령)
- **Milestone**: 개발·허가·상업화 합산 최대 **$9.85B**
- **Total biobucks**: 최대 **$10.5B**
- **Royalty**: 각 승인 제품 판매 시 **double-digit royalties**
- **3-tier rights 구조**:
  - 4개 프로그램: 미국·유럽 공동개발·공동판매, 중국은 Innovent 보유
  - 4개 프로그램: 중국 제외 글로벌 Pfizer exclusive
  - 4개 프로그램: 전 세계 Pfizer exclusive (개발 비용 전액 Pfizer 부담)
- **단계별 역할**: Innovent → Phase 1 lead; Pfizer → Phase 2 이후 글로벌 개발·허가·상업화

**왜 이 딜인가**:
- Pfizer는 Paxlovid 매출 급감 이후 oncology 포트폴리오 재건이 최우선 과제 (2023년 Seagen 인수 $43B으로 ADC 전문성 확보 이후 파이프라인 심화 필요)
- Innovent는 PD-1 항체(sintilimab, 중국 시장 점유율 1위), ADC, multispecific antibody에서 검증된 R&D 역량 보유
- 중국의 빠른 임상 환자 모집·저비용 Phase 1 실행 + Pfizer의 글로벌 commercial infrastructure = 위험 분산 최적화 구조
- "5개월 스프린트"로 협상 완료 — Pfizer의 전략적 urgency 반영

**밸류에이션 근거**:
- 12개 프로그램 × 평균 biobucks ~$875M/program
- Phase 1 oncology ADC PoS(승인까지 누적): ~10-12% → 기대 milestone PV: ~$87.5M/program
- Per-program rNPV(milestone+royalty): ~$300-400M → 12개 portfolio rNPV ~$3.6-4.8B
- Pfizer 실질 현금 리스크: $650M upfront — 나머지는 성과 연동(milestone), Pfizer에 유리한 리스크 구조

**비교 딜**:
| 딜 | Upfront | Total | Stage | Per-Asset biobucks |
|---|---|---|---|---|
| Pfizer-Innovent (2026) | $650M | $10.5B | Phase 1 (12개 에셋) | ~$875M |
| BMS-SystImmune BL-B01D1 (2024) | $800M | $8.4B | Phase 2 (1개 에셋) | $8.4B |
| AZ-Daiichi Dato-DXd 글로벌 (2023) | $2B | ~$6B | Phase 3 (1개 에셋) | $6B |

→ 성숙도 낮을수록 per-asset biobucks 작아지는 정상적 패턴

**리스크 요인**:
- Phase 1 에셋 12개 → 임상 리스크 매우 높음 (누적 PoS 10-12%)
- 미중 규제 리스크: Biosecure Act 향방, 미국 내 중국 바이오 파트너십 규제 강화 가능성
- Pfizer 내부 12개 프로그램 자원 경쟁 → 우선순위 밀릴 경우 개별 에셋 개발 지연

### 2-2. AbbVie × Apogee Therapeutics: $10.9B 인수 — IL-13 차세대 HLE 항체 확보

**딜 개요**:
AbbVie가 Apogee Therapeutics를 주당 $135.11, 총 equity value **$10.9B**에 전액 현금으로 인수 발표. **2026년 6월 22일** (불과 4일 전!) 발표. Q3 2026 클로징 예정.

**딜 구조**:
- **Type**: All-cash M&A (전액 현금 인수)
- **Total equity value**: **$10.9B**
- **Lead asset**: zumilokibart (APG777) — anti-IL-13 half-life extended (HLE) monoclonal antibody for atopic dermatitis (Phase 2) & asthma
- **Pipeline**: atopic dermatitis·천식·기타 면역학 적응증 multiple clinical-stage candidates 포함

**왜 이 딜인가**:
- AbbVie의 Humira 바이오시밀러 침식으로 immunology 매출 방어 필수 — Skyrizi·Rinvoq 성장으로 일부 상쇄 중이나 next wave 필요
- Dupixent(dupilumab, Sanofi/Regeneron)이 IL-4/IL-13 이중 차단으로 atopic dermatitis 왕좌 차지 중 (~$16B/년)
- Zumilokibart는 **IL-13 선택적 차단 + HLE 기술**(반감기 연장으로 월 1회 또는 격월 투여 가능) → Dupixent 대비 편의성 우위 포지셔닝 가능
- Phase 2 결과: 16주 시점 약 2/3 환자에서 현저한 피부 호전(EASI 75 수준), 소양증·삶의 질 지표 개선

**밸류에이션 근거**:
- Atopic dermatitis 글로벌 시장 2030년 $20B+ 예상 (Dupixent가 현재 $16B 점유)
- IL-13 selective HLE mAb의 peak sales 잠재력: $3-5B (Dupixent 점유율의 20-30% 잠식 가정)
- rNPV 역산: $10.9B ÷ PoS(Phase 2→시판, ~30-40%) = implied undiscounted peak value ~$27-36B → peak sales 가정과 consistent
- 비교: Sanofi가 Dupixent Phase 3 당시 Regeneron에 지불한 금액 vs. 현 Dupixent 매출 = 고위험 고수익 공식

**리스크 요인**:
- Dupixent와의 Head-to-Head 비교 임상 필요 여부 → 없으면 payer/의사 설득 어려움
- AbbVie 자체 IL-13 자산(skrisesti 등)과 내부 포트폴리오 cannibalization 가능성
- Phase 3에서 Phase 2 결과 재현 리스크 (n이 작은 Phase 2 데이터)
- $10.9B는 Phase 2 에셋으로서 최상위권 프리미엄 — 실패 시 대규모 write-off

### 2-3. GSK × Hengrui Pharma: $12.5B 다품목 옵션 딜 — COPD 신약 포함 12개 에셋

**딜 개요**:
GSK가 중국 Hengrui Pharma로부터 HRS-9821(PDE3/4 이중 억제제, COPD) 포함 최대 12개 에셋의 중국 제외 글로벌 개발·판매 권리를 옵션 구조로 확보. $500M upfront, milestone 합산 ~$12B (2025년 체결, 2026년 실행 중).

**딜 구조**:
- **Upfront**: **$500M** (12개 프로그램 배분)
- **옵션 구조**: Hengrui가 Phase 1 완료 데이터 생성 후 GSK에 옵션 제공 → GSK가 프로그램별 개발 참여 여부 독립 결정
- **총 milestone**: 전 프로그램 합산 ~$12B
- **Royalty**: 글로벌 순매출(중국 4개 지역 제외) tiered royalties
- **Lead asset HRS-9821**: COPD add-on 유지 치료용 PDE3/4 이중 억제제 — Phase 2 중국 완료, 글로벌 Phase 3 준비 중

**왜 이 딜인가**:
- GSK는 respiratory 글로벌 1위이나 COPD 차세대 small molecule (PDE3/4 계열) 파이프라인이 취약
- HRS-9821: PDE3(기관지 확장) + PDE4(항염증) 동시 억제 → 단일 tablet로 bronchodilation + inflammation 두 가지 효과 → 기존 ICS/LABA/LAMA 3제 요법 보완 또는 대체
- **Option 딜의 전략적 매력**: $500M upfront으로 12개 에셋에 대한 call option 확보 → 개별 Phase 1 데이터 보고 후 선별 행사 → downside 제한, upside 유지

**BD 포인트 — Option Deal 구조 이해**:
- Full acquisition 대비 option deal은 "정보 비대칭 해소 후 결정" 전략
- GSK는 Hengrui가 Phase 1 데이터(인체 안전성·PK·초기 효능 신호)를 생성하는 동안 $500M의 자금과 리스크만 부담
- Phase 1 성공 후 option 행사 시 추가 지불 → 리스크 조정된 진입

**리스크 요인**:
- Phase 1 에셋 다수 → attrition 높음
- PDE4 억제제 계열 부작용(구역감, 두통, 심혈관계) 모니터링 필요
- 미중 규제 긴장: Biosecure Act 등 향방에 따른 불확실성

### 2-4. OliX × Eli Lilly: $630M 한국 RNAi — MASH·비만 이중 효과 siRNA 글로벌 딜

**딜 개요**:
한국 **OliX Pharmaceuticals**의 RNAi 기반 MASH 치료 후보물질 **OLX75016**(MARC1 타겟 siRNA)의 글로벌 개발·판매권을 Eli Lilly에 라이선스아웃. 총 계약 규모 약 910억원(~$630-660M). Phase 1 진행 중 (호주, 2024년 2월 시작).

**딜 구조**:
- **Total deal value**: ~$630-660M (upfront + milestone)
- **Upfront**: 미공개 (Phase 1 완료 지원금 포함)
- **플랫폼**: OliX 자체 개발 asymmetric siRNA (asiRNA) 기술 — 기존 siRNA 대비 off-target effect 감소, 간 선택적 LNP 전달 시스템 결합
- **권리**: Lilly가 글로벌 개발·상업화 exclusive rights

**OLX75016 메커니즘**:
- **MARC1 (Mitochondrial Amidoxime Reducing Component 1)**: GWAS(유전체 연관 연구)를 통해 발견된 간 보호 유전자. MARC1 기능 상실(loss-of-function) 변이를 가진 사람들은 MASH·지방간·비만에 강한 보호 효과를 보임.
- OLX75016은 MARC1 발현을 억제하는 siRNA → MARC1 발현 저하 → 간 지방 축적 감소 + 체중 감소 효과
- 원숭이 실험: 세마글루타이드(Wegovy 성분)와 병용 시 **체중 감량 상승 효과 + 요요 방지** 확인 → GLP-1 병용 시너지 플랫폼으로서의 포지셔닝

**밸류에이션 근거**:
- MASH 글로벌 시장: Resmetirom(Rezdiffra, Madrigal) FDA 승인(2024) 이후 급성장, 2030년 ~$10-15B
- Phase 1 RNAi 에셋으로 $630M은 높은 편 → MASH+비만 dual indication 프리미엄 반영
- Lilly의 GLP-1(tirzepatide) 플랫폼과 combination therapy 시너지 가치 포함
- 참고: Alnylam inclisiran(PCSK9 siRNA) → Novartis $9.7B 딜 (Phase 3 성숙 에셋) — OliX는 훨씬 earlier stage임에도 상당한 밸류

**BIO USA 2026 맥락**:
- OliX는 BIO USA 2026(San Diego, 2026.06.22-25)에 한국 대표 RNAi 기업으로 참가
- OliX CEO는 2H 2026에 MASH Phase 2 데이터 및 비만 신약(OLX501A, ALK7 타겟) BD 활동 강화 예고
- Lilly가 한국 바이오텍에 $500M 이상 투자한 대표 사례 — K-bio 글로벌 출격 기반 강화

**리스크 요인**:
- Phase 1 단계 — 사람에서의 안전성·유효성 미입증
- siRNA 간 선택적 전달 체계(LNP) 효율·안전성 의존
- MASH 경쟁 치열화: Resmetirom(승인), GLP-1, 이중/삼중 작용제들 시장 선점 가속

### 📌 Section 2 Takeaways

**오늘 딜에서 배울 수 있는 딜 구조 패턴:**
> **"Phase 1 Option Deal" 패턴** — GSK-Hengrui처럼, 중국·한국 바이오텍이 Phase 1을 완료하면 빅파마가 option을 행사해 글로벌 권리를 가져가는 구조. Upfront는 낮지만 Phase 1 데이터 생성 후 option premium + milestone + royalty로 극대화. 한국 biotech가 벤치마크할 수 있는 가장 실용적인 out-licensing 모델: "Phase 1까지 자력으로, 그 이후 option 제공".

**비슷한 에셋을 가진 한국/아시아 회사 벤치마크:**

**BMS 관점에서 이 딜들의 의미:**
- **Pfizer**가 $650M upfront+12개 중국 ADC 에셋 확보 → BMS도 중국·한국 sourcing 전략 재검토 시급 (BIO USA 2026이 바로 그 기회)

## Section 4: 오늘의 BD 역량 심화 (BD Competency Deep Dive)

### 🎓 오늘의 BD 스킬: [[rNPV]] — Risk-Adjusted NPV 계산법과 딜 밸류에이션 역산

---

**1. 오늘의 BD 스킬**: [[Deal Valuation]] — rNPV 계산 및 Pfizer-Innovent $10.5B 딜에 적용한 역산 검증

---

**2. 실제 사례**: Pfizer × Innovent $10.5B 딜 (12개 oncology 프로그램, 2026년 5월)

Pfizer는 Phase 1 단계 에셋 다수를 포함한 12개 프로그램에 $650M upfront를 지불하고 최대 $10.5B을 약속했다. BD 담당자라면 "이 딜은 합리적으로 가격이 책정되었는가?"를 [[rNPV]]로 검증해야 한다.

---

**3. 개념 설명**:

**NPV (Net Present Value)**란 미래 현금흐름(신약 매출 - R&D/판매비용)을 현재가치로 환산한 것. 신약 NPV = 향후 peak sales 기반 현금흐름 × 할인율로 PV 계산.

**rNPV (Risk-adjusted NPV)**는 임상 성공 확률(Probability of Success, PoS)을 각 단계별로 곱하여 리스크를 반영한 현재가치:

```
rNPV = NPV × PoS(현 단계 → 시판 허가)
```

**단계별 oncology 평균 PoS (산업 표준)**:
| 단계 | 다음 단계 전환 성공률 |
|------|-------------------|
| Phase 1 → Phase 2 | ~60% |
| Phase 2 → Phase 3 | ~35% |
| Phase 3 → NDA 제출 | ~60% |
| NDA → FDA 승인 | ~85% |
| **Phase 1 → 승인 누적 PoS** | **~10-12%** |

---

**4. 실전 적용**: Pfizer-Innovent 딜 rNPV 역산 (5단계)

**Step 1: 개별 에셋 평균 biobucks 계산**
```
Total biobucks = $10.5B ÷ 12 programs = $875M per program
Upfront 배분 = $650M ÷ 12 = $54M upfront per program
```

**Step 2: Phase 1 에셋 기대 milestone PV 계산**
```
PoS(Phase 1 → 승인) = 10%
기대 milestone PV = $875M × 10% = $87.5M per program (risk-adjusted)
```

**Step 3: Royalty stream 기대 PV 추정**
```
가정: peak sales $1-2B per product × royalty rate 12% × 할인·기간 보정
Royalty stream rNPV ≈ $150-250M per program
```

**Step 4: Per-program rNPV 종합**
```
Per-program rNPV = 기대 milestone PV + royalty stream rNPV
                 = $87.5M + $150-250M
                 ≈ $240-340M per program (risk-adjusted)
```

**Step 5: Portfolio 관점 합리성 검증**
```
12개 portfolio rNPV = 12 × $240-340M = $2.9-4.1B
Pfizer 실질 현금 지출(upfront) = $650M
→ 기대 portfolio rNPV($2.9-4.1B) >> 현금 upfront($650M)
→ 나머지 $9.85B은 performance-gated milestone → 리스크 분산된 합리적 딜
```

**비교 분석으로 Cross-check**:
| 딜 | Upfront | Total | Stage | Per-Asset 논리 |
|---|---|---|---|---|
| Pfizer-Innovent (2026) | $650M | $10.5B | Phase 1 (12개) | $875M/program × PoS 10% = $87.5M EV → 적정 |
| BMS-SystImmune BL-B01D1 (2024) | $800M | $8.4B | Phase 2 (1개) | 단일 Phase 2 에셋 → higher PoS (~25%) 반영 |
| AZ-Daiichi Dato-DXd (2023) | $2B | $6B | Phase 3 (1개) | Phase 3 에셋 → PoS ~50% → $3B EV → 적정 |

→ **성숙도 낮을수록 per-asset biobucks가 낮아야 정상** — Pfizer-Innovent 딜은 Phase 1 다품목 portfolio bet으로서 합리적 가격
